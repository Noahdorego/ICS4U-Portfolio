Advanced JS: Natural Simulations

Natural Simulations was a very eye-opening module for me. It displayed examples of the vital connection between math and coding. Specifically, learning about vectors and forces helped me understand how realism can be added to games/simulations. The whole module was advanced: nothing was "dumbed down" or explained for children. It felt like a real step in my future of coding.

AP®︎/College Computer Science Principles

I completed the entire AP Computer Science Priciples course, and I'm glad I did. Outside of coding, I also want to know exactly how the digital world works. This course provided me with the base knowledge I need in several topics, and each unit was relevant.

The two units which I appreciated the most are Digital Information and The Internet. Digital Information helped me understand how data is stored and transferred, and what formats are most effective. The Internet showed me the basics of networks, transporting packets and web protocols, which I wanted to know. This was a super useful course for me.

Intro to JS: Drawing & Animation

I've completed and recompleted this module 3 times. Yet, each time teaches me something new or solidifies a coding concept in my head. The initial part of the module is review, as the basics are taught, but I focused on the object-oriented part of the module. OOP is a very efficient way of coding, if it is implemented correctly. This module gave me the chance to improve my knowledge and skills in the subject. I learned about object inheritance, classes, attributes and more. I will continue learning about OOP in the future, as it is an important concept to know.

Advanced JS: Games & Visualizations

Since I have experience in game development, this was a perfect module for me. I learned about game mechanics, scenes, movement and more. It also expanded on the object oriented programming, which allowed me to practice with it.

Codecademy ("Create Video Games with Phaser.js Skill Path")

This was the skill path I chose to do in Codecademy. There were many that I was interested in, but I chose this one to make basic JS concepts more concrete in my head. However, I used this module to observe how Codecademy teaches and see if I'd want to continue using the platform. I can say with confidence that I will use Codecademy in my future, as it demonstrates an effective method of teaching, and introduces you to coding in the real world.

Intro to HTML/CSS: Making Webpages

This was a good refresher on HTML and CSS. It helped me to develop better practices and use different features.

HTML/JS: Making Webpages Interactive

This was useful because it introduced something that I had never even heard of, the DOM. The Document Object Model represents the document in objects and nodes, and allows you to make dynamic changes to the program in real time. This is something that I want to learn more about, as it is a side of JS that I've never seen.
